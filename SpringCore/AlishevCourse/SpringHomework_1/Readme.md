## Задание 
1) Создайте список (или массив) из 3 песен в классе  **ClassicalMusic** и **RockMusic**(по 3 песни на каждый жанр)
2) В **MusicPlayer** должны внедряться бины ClassicalMusic и RockMusic
3) Создать Enum c двумя жанрами музыки - **CLASSICAL** и **ROCK**   
4) Переделать метод **playMusic()** так,чтобы он принимал на вход этот Enum
5) В зависимости от значения этого Enum'a (CLASSICAL или ROCK) метод **playMusic()** должен воспроизвести случайную классическую или рок песню (использовать класс Random).
